## 🌟 The Future of Data Interaction

Welcome to **DataSenseAI**! This phenomenal platform redefines how you interact with both complex local files and live financial systems. By leveraging state-of-the-art AI, DataSenseAI empowers you to query, analyze, and extract knowledge using pure natural language. Whether you're an analyst trying to find the next stock pick or just a user asking questions about a dense spreadsheet—simply ask, and DataSenseAI delivers!

## 🔮 Capabilities Driven By Specialized AI Skills

DataSenseAI isn't just a basic reader; it's equipped with expert-level AI skills turning it into a specialized **Stock Research & Financial Forecast Analyst**.

### 📈 Intelligent Financial & Stock Analysis
- **Live Market Scraping:** Scrapes data from top-tier sources like Moneycontrol, ET Markets, and Screener.in to answer your questions with completely live context.
- **Stock Forecasting:** Extracts live stock prices, analyst target projections, revenue, and dividend histories on the fly.
- **Expert Analysis Engine:** The built-in AI acts as an expert financial research analyst, synthesizing data points—from quarterly earnings to historical profit—to give actionable, high-quality answers.

### 🗂️ Universal File Intelligence
- **Seamless Spreadsheet Parsing:** Instantly query and cross-reference across CSV and Excel documents.
- **PDF Document Intelligence:** Read, summarize, and synthesize critical information from dense PDF reports in seconds.
- **Live Online Context:** Connect offline document knowledge with dynamically retrieved live web insights.

## 🛠️ Run the Revolution Locally

Want to experience the magic on your local machine? Getting started is completely frictionless!

1. **Clone the repository**
   ```bash
   git clone https://github.com/HustlerKrishna1/DataSenseAi.git
   cd DataSenseAi
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch the development server**
   ```bash
   npm run dev
   ```

Open up your browser and navigate to `http://localhost:3000` to start querying your data!
