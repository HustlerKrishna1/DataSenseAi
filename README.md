# DataSenseAI

DataSenseAI is an AI-powered tool that allows you to query data seamlessly using natural language. It supports querying directly from various file formats including:
- CSV
- Excel
- PDF

Additionally, it can integrate with and query information from online web sources.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
